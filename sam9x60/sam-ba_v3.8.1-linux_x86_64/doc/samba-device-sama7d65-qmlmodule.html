    The SAMBA.Device.SAMA7D65 module contains an implementation of the SAMBA::Device type for the SAMA7D65. device.

    SAMA7D65

    Contains chip-specific information about SAMA7D65 device

    SAMA7D65CURIOSITY

    Contains a specialization of the SAMA7D65 QML type for the SAMA7D65 FPGA plateform

    SAMA7D65Config

    Contains configuration values for a SAMA7D65 device

    SAMA7D65DDR3EB

    Contains a specialization of the SAMA7D65 QML type for the SAMA7D65 DDR3 EB plateform

    SAMA7D65SIPEB

    Contains a specialization of the SAMA7D65 QML type for the SAMA7D65 SIP plateform
